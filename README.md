# NUR AL NAHAR Perfumes L.L.C — Website

Premium, modern website for a next‑generation Dubai perfume house.

## Tech Stack
- Next.js (App Router)
- TypeScript
- Tailwind CSS
- shadcn/ui-style components (Radix UI + Tailwind)
- Framer Motion for smooth page transitions

## Getting Started
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
npm run start
```

## Config
Set your official phone/WhatsApp number via env:

- `NEXT_PUBLIC_CONTACT_PHONE="+971 50 123 4567"`

You can copy `.env.example` to `.env.local`.
